Get started with our webserv

This project is here to make you write your own HTTP server Using C++98.
You will be able to test it with a real browser. HTTP is one of the most used protocol on internet.
Knowing its arcane will be useful, even if you won't be working on a website.

  • Execution :

    make && ./webserv path/configFile.conf

Stuff Implemented In This Web Server :

  • - MultiPort listening
  • - HTTP Method : GET -- POST -- DELETE
  • - Multi FastCGI : Go -- Python -- PHP
  • - Cookies and Session Managing
